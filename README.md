- Node.js
- React JS
- SQLite
- REST API
- Responsive UI

## Project
Web application for the curious. Discover details about your name, generation, and your zodiac sign.

## Getting Started frontend
- Install node.js
- Run npm install
- Run yarn Start

## Getting Started backend
- Install .NET framework
- Run dotnet build
- Run dotnet run
